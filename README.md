A thermometer project
=====================

Displays temperature in large font, humidity and heat index in a tiny font.

Parts used:
-----------
- Arduino Nano v3
- DHT11 temperature & humidity sensor
- 128x32 OLED display (SSD1306)
- half-size breadboard
